# docker-riemann

[![Docker Pulls](https://img.shields.io/docker/pulls/mashape/kong.svg)](https://hub.docker.com/r/neetjn/riemann)

Docker image for Riemann network monitoring system version 0.2.11.

```

```
FROM neetjn/riemann:latest
